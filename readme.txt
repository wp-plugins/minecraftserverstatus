MinecraftServerStatus widget is displaying MC server status and player count

== Description ==
Widget with various information about your minecraft server.

You can display these information:

 * Server name
 * Status (online/offline)
 * IP address / hostname
 * port
 * response time (ping)
 * player count (current / maximum)

and more features are planned

== Installation ==
1. Upload all files to the `/wp-content/plugins/` directory
1. Activate the plugin through the \'Plugins\' menu in WordPress
